# Classic ML

Implementing classic machine learning algorithms from scratch and with scikit-learn, as a learning project.

## Structure
- `datasets/` — sample datasets used across models
- `linear-regression/` — linear regression from scratch + sklearn
- More model folders added as I progress through each topic

## Setup

```bash
python -m venv .venv
.venv\Scripts\activate      # Windows
source .venv/bin/activate   # Mac/Linux
pip install -r requirements.txt
```
